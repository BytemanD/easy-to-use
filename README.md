# any-code
The lib of some python utils

### install lib

> ubuntu

```
apt-get install glibc gcc python2-dev
```
> centos

```
yum install glibc gcc python2-devel
```


### install requirements
```
wget https://bootstrap.pypa.io/pip/2.7/get-pip.py
python2 get-pip.py
pip2 install -r requirements.txt
```
